# Portafolio personal — Diego Medina

Sitio web personal donde muestro los proyectos en los que he trabajado, mi experiencia y las tecnologías que uso. Está disponible en español e inglés.

Ver en vivo: https://portafolio-sandy-phi-49.vercel.app/

---

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- next-intl para internacionalización (ES / EN)

---

## Estructura

El sitio tiene las siguientes secciones:

- Presentación y descripción profesional
- Proyectos filtrados por categoría
- Experiencia laboral
- Stack de tecnologías
- Preguntas frecuentes
- Contacto

---

## Instalación local

```bash
npm install
cp .env.local.example .env.local
npm run dev
```

El sitio queda disponible en `http://localhost:3000`.
